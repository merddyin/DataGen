# Capability Overview

Current DataGen builds cover more than just identities and devices. The platform now spans a broad synthetic enterprise graph.

## Major capability families

- organizational and geography generation
- people, accounts, groups, identity stores, and containers
- infrastructure, devices, servers, network assets, and telephony
- applications, application services, cloud tenants, and dependencies
- repositories, databases, file shares, collaboration sites, and tabs
- policies, targeting, and access-control evidence
- observed data and source-system snapshots
- CMDB configuration items and realistic source-facing drift
- external ecosystem and counterparty relationships

## Why that breadth matters

This allows one generated world to support multiple consumers at once:

- lab population
- discovery validation
- CMDB reconciliation testing
- repository and collaboration demos
- export-driven downstream integration

The rest of the capability pages break these surfaces into more concrete usage-oriented summaries.
